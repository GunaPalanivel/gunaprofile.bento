# Guna Portfolio

A modern, hand-crafted portfolio website built with pure HTML and CSS. Features a bento grid layout with playful, creative design elements.

## Tech Stack

- **HTML5**: Semantic markup
- **CSS3**: Grid layout, custom properties, animations
- **No frameworks**: Pure vanilla code

## Project Structure

```
gunaprofile/
├── index.html          # Main HTML file
├── css/
│   ├── main.css        # Main stylesheet (imports all modules)
│   ├── reset.css       # CSS reset
│   ├── grid.css        # Grid layout system
│   ├── typography.css  # Font and text styles
│   ├── components.css  # Reusable components
│   └── responsive.css  # Media queries
├── fonts/              # Custom fonts
├── assets/             # Images and icons
└── README.md
```

## Development

Open `index.html` in a browser to view the portfolio.

Hand-crafted code with attention to:

- Modular, maintainable CSS
